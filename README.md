# project-A-FE

동아리음 프론트엔드

## 폴더 구조

```
- public
- src

  - api # api 함수
  - assets # 이미지, 아이콘
  - components # 공통 컴포넌트
  - context # 전역 상태
  - hooks # 공통으로 사용하는 커스텀훅
  - pages # 페이지 컴포넌트
  - styles # 스타일
  - utils # 유틸 함수

  - App.tsx
  - Router.tsx
  - index.tsx
```

## 커밋 컨벤션

| 타입     | 설명                    |
| -------- | ----------------------- |
| feat     | 새로운 기능 추가        |
| fix      | 버그 수정               |
| docs     | 문서 수정               |
| refactor | 코드 리팩토링           |
| remove   | 불필요한 코드/파일 삭제 |
| test     | 테스트 코드             |
| chore    | 설정, 빌드, 기타 잡일   |
| style    | 스타일 수정             |

```
feat: 기능 수정
chore: ~~
```
